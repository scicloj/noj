3  Turning tensors into images

(ns image
  (:require [tech.v3.tensor :as tensor]
            [tech.v3.datatype.functional :as fun]
            [scicloj.noj.v1.vis.image :as vis.image]))

The vis.image/tensor->image function can help turning dtype-next’s tensors into Java BufferedImage objects.

Eventually, this function may find itself in dtype-next itself, see Issue #92 there.

You may see this function in action in the Clay & Noj demo: image processing at the Clojure Data Scrapbook and in the related video.

(-> (for [i (range 100)]
      (range 100))
    tensor/ensure-tensor
    (fun/* 400)
    (vis.image/tensor->image :ushort-gray))
